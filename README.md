# 🍽️ Lumiere Restaurant Web App

## 🌐 Live Demo

https://lumiere-frontend.onrender.com

## 📌 About

A full-stack restaurant web application where users can browse menu items, book tables, and make secure online payments.

## 🚀 Features

* User Authentication (Login/Register)
* Table Reservation System
* Dynamic Menu Display
* Online Payments (Stripe)
* User Dashboard

## 🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Node.js, Express
* Database: MongoDB
* Payment: Stripe
